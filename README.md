# 🎯 EtalonBot

Telegram-бот для персонального развития: помогает пользователю пройти путь от «Точки А» (текущее состояние) к «Эталонной версии себя».

## 📦 Назначение

- 🤖 Персональный ИИ-коуч в Telegram
- 🗺️ Диагностика «Точки А» — анализ текущего состояния
- 🎯 Построение траектории развития к эталонной версии
- 💬 Интерактивные сессии работы с ограничивающими убеждениями

## 🚀 Как использовать

1. **Найти бота**: [@etalon_bot](https://t.me/etalon_bot) в Telegram
2. **Начать диагностику**: `/start` → ответить на вопросы о текущем состоянии
3. **Получить план**: бот сформирует траекторию развития
4. **Работать пошагово**: интерактивные сессии и микротренировки

## 🛠️ Технический стек

- **Язык**: Python 3.10+
- **Фреймворк**: python-telegram-bot
- **ИИ-интеграция**: Claude API
- **Хранение**: SQLite / PostgreSQL
- **Развёртывание**: Vercel / VPS
